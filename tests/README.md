# susieR Testing Framework

This directory contains the comprehensive test suite for the susieR 2.0 package, with **898 total tests** ensuring code correctness, stability, and consistency with the reference implementation.

## File Organization

- `testthat/`: Directory containing test files
    - `helpers`: Folder containing helper functions used in testing for simulating data, assigning attributes, and more
    - `reference`: Reference tests ensuring consistency with original susieR 1.0 implementation
    - `unit`: Unit tests validating correctness and stability for all functions in susieR 2.0
